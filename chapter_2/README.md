## 条件分岐 (if文)

### バイナリファイルの作成

```
$ clang++-10 -Wall -Wextra -O3 -S -masm=intel <ファイル名>
```

### バイナリファイルを読む

```
$ cat if.s | c++filt | less
```

コードとバイナリファイルを分割表示してくれる

### おまけ

代入演算子について

```
int n{1};

int x{n = 3}; //式文
// 作用: n = 3 で 3 を返す
// 副作用: n に 3 を代入
```

=> 変数に値を代入するという処理は代入演算子の副作用を使っている
